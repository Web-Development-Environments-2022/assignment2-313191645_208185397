    About The PACMAN Game

    This game was developed by Ilan Korol and Ben Aidlin
    as part of a web development course in the third year of
    SISE studies in Ben Gurion University.

    Please feel free to log in or sign in, then control the settings,
    and finally play. You can adjust any setttings during the game itself, and restart the game by pressing "Play".

    We used JQuery to add all of the onClick, onPress, when doc ready, etc.. functionallity, and also to validate the sign in details.
    We used the template we recieved but did modify it quite a lot to make it ours and unique.
    Some of our difficulties were fitting the sizes of the images to the canvas, trying to keep the code clean,
    and also, the fact that we saw we need to add difficulties here after we already submitted. :)















    Changes will be updated after you press "Play"
